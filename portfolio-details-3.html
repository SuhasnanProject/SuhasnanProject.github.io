Project Description

In an era where cardiovascular health is of paramount importance, the project titled "Dashboard of Heart Disease Indicators Based on Age and Gender (2020)" aims to provide a comprehensive analysis of critical factors influencing heart health. This initiative delves into the intricate interplay between age, gender, and heart-related indicators, shedding light on patterns and trends that can inform both individuals and healthcare professionals.

The dataset utilized for this project was sourced from Kaggle, a prominent platform for data science enthusiasts. This rich dataset encompasses a wide array of variables, allowing for a nuanced exploration of heart disease indicators. By leveraging Python for data manipulation, SQL for database querying, and Looker Studio for advanced data visualization, the project seamlessly combines these powerful tools to derive meaningful insights.

The integration of these technologies not only facilitates a meticulous analysis of the dataset but also enables the creation of an interactive and visually engaging dashboard, enhancing accessibility and comprehension for diverse audiences.
